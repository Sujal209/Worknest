# WorkNest 🐼

WorkNest is a centralized platform designed to connect **Gen Z students and freshers with real-world opportunities** such as internships, volunteering roles, and short-term projects.

The platform bridges the gap between **emerging talent and organizations** by making opportunity discovery simple, accessible, and skill-focused.

Our goal is to empower young individuals to -work, learn, and contribute seamlessly.

---

# 🚀 Features

### Dynamic Profiles
Create a profile that highlights your skills, projects, and contributions, not just a traditional resume.

### 1-Click Apply
Apply instantly to opportunities using your WorkNest profile and track your application progress.

### Smart Matching
WorkNest recommends opportunities based on skills and interests.

### Dedicated Dashboards

Student / Fresher Dashboard
- View opportunities
- Track applications
- Update profile and skills

Recruiter Dashboard
- Post opportunities
- Manage applicants
- Communicate with candidates

### Responsive Design
The platform is fully responsive and works smoothly on desktop and mobile devices.

# 📦 Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/Shravanidhuri/WorkNest.git
cd WorkNest
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Backend Server

```bash
node server.js
```

The server will start on:

```
http://localhost:5000
```

---

# 💻 Run the Frontend

You can run the frontend in two ways:

### Option 1 — Open Directly
Open `index.html` in your browser.

### Option 2 — Use a Local Server

```bash
npx serve .
```
